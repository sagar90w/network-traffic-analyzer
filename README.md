# Network Traffic Analyzer

A simple Python-based network traffic analyzer built using **Scapy**.  
It captures packets, displays important details, and helps beginners understand basic packet inspection.

## Features
- Live packet capture
- Protocol detection (TCP, UDP, ICMP)
- Source & destination IP extraction
- Port detection
- Simple CLI interface

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the analyzer:
   ```
   python main.py
   ```

## For Learning Purpose Only
Use ethically and on your own network.
